# Ressursar til Python, mikrodatamaskiner og mikrokontrollerar

Ein samling med opplegg, kodeeksempel, oppgåver og løysingsforslag for ulike prosjekt som brukar Raspberry PI, Raspberry PI Pico, Arduino, ESP32 og liknande plattformer.

Vi snakker om både mikrodatamaskiner (som Raspberry PI) og mikroontrollerar (som Raspberry PI Pico og Arduino). Det er ikkje alltid like lett å vite kva som er kva - det viktigaste er at ein kan bruke dei til å lage spennande prosjekt, og lære seg programmering og elektronikk på ein praktisk måte. Dersom me likevel skal klassifisere ein eining, så kan me forenkla sjå det slik:
- `Mikrodatamaskin`: Køyrer eit operativsystem som kan køyre fleire program samtidig, og typisk ha meir minne og prosessorkraft enn ein mikroontroller. Skrur du på strømmen, så startar den opp og køyrer operativsystemet, og du kan køyre ulike program og gjere mange ting samtidig.
- `Mikroontroller`: Er ofte enklare og meir spesialiserte, og brukar mindre energi. Skrur du på strømmen, så startar den opp og køyrer det programmet som er lasta inn, og har ikkje eit operativsystem i tradisjonell forstand.

Kom gjerne med innspel til forbetringar av koden.

Innhald:
- Opplegg rundt [Raspberry PI](opplegg%20-%20raspberry%20pi.md), og tilhøyrande utstyr, som kamera, SenseHAT, EnviroHAT, sensorar og liknande.
- Kodeeksempel for  [Raspberry PI Pico](opplegg%20-%20raspberry%20pico.md), og tilhøyrande utstyr, som lyslenker og liknande.
- Kodeeksempel for [Arduino](opplegg%20-%20arduino.md), og tilhøyrande utstyr, som sensorar for ein havbøye, motorar og liknande.
- Kodeeksempel for [verstasjonen EnviroHAT](./verstasjon/) for Raspberry PI.
- Meir kjem til...
