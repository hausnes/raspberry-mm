# Ressursar til Python, mikrodatamaskiner og mikroontrollerar

Ein samling med opplegg, kodeeksempel, oppgåver og løysingsforslag for ulike prosjekt som brukar Raspberry PI, Raspberry PI Pico, Arduino, ESP32 og liknande plattformer.

Kom gjerne med innspel til forbetringar av koden.

Innhald:
- Opplegg rundt [Raspberry PI](opplegg%20-%20raspberry%20pi.md), og tilhøyrande utstyr, som kamera, SenseHAT, EnviroHAT, sensorar og liknande.
- Kodeeksempel for  [Raspberry PI Pico](opplegg%20-%20raspberry%20pico.md), og tilhøyrande utstyr, som lyslenker og liknande.
- Kodeeksempel for [Arduino](opplegg%20-%20arduino.md), og tilhøyrande utstyr, som sensorar for ein havbøye, motorar og liknande.
- Kodeeksempel for [verstasjonen EnviroHAT](./verstasjon/) for Raspberry PI.
- Meir kjem til...

Det ligg ellers ein del kode for å lære seg programmeringsspråket Python i [denne mappa](./python-kurs/).
